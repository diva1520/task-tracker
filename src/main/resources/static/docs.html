API Reference

Documentation for Task Tracker Application Endpoints

Authentication
POST /auth/login
Authenticate user and retrieve a JWT token.
Request Body
{
  "username": "user1",
  "password": "password123"
}
Response
{
  "token": "eyJhbGciOiJIUzI1NiJ9..."
}
Database Context
Reads: users (Verify credentials)
Writes: user_login_audit (Log login event)
POST /auth/logout
Log out the current user and record logout time in audit logs.
Headers
Authorization: Bearer
Response
200 OK (No Body)
Database Context
Updates: user_login_audit (Set logout time & duration)

POST /user/tasks/log-work
Log work hours for a specific task.
Validation Rules
  • Time Overlap: The system validates that the new time entry does not overlap with any existing logs for the user on the same day.
  • Sequence: End time must be after Start time.
Request Body
{
  "taskId": 101,
  "startTime": "2024-01-20T10:00:00",
  "endTime": "2024-01-20T12:00:00",
  "comment": "Fixed bugs",
  "status": "IN_PROGRESS"
}
Response
"Work log added successfully"
Database Context
Reads: work_logs (Check for overlaps)
Inserts: work_logs (Record time entry)
Updates: task (Update total worked minutes)

Database Schema
Table: users
Column Type Description
id BIGINT Primary Key
username VARCHAR Unique login ID
password VARCHAR Encrypted password
email VARCHAR User email
role VARCHAR ROLE_ADMIN or ROLE_USER
active BOOLEAN Account status
Table: task
Column Type Description
id BIGINT Primary Key
title VARCHAR Task title
description TEXT Detailed description
status VARCHAR TO_DO, IN_PROGRESS, etc.
user_id BIGINT Assigned User (FK)
assigned_by BIGINT Admin who assigned
created_at DATE Creation date
due_date DATE Deadline
total_worked_minutes BIGINT Aggregated time
Table: work_logs
Column Type Description
id BIGINT Primary Key
task_id BIGINT Related Task (FK)
user_id BIGINT User (FK)
start_time DATETIME Log start
end_time DATETIME Log end
duration_minutes BIGINT Calculated duration
comment VARCHAR Work description
Table: user_login_audit
Column Type Description
id BIGINT Primary Key
user_id BIGINT User ID
login_time DATETIME Login Timestamp
logout_time DATETIME Logout Timestamp
ip_address VARCHAR User IP
status VARCHAR ACTIVE/LOGOUT
Table: task_detail
Column Type Description
id BIGINT Primary Key
task_id BIGINT Related Task (FK)
status VARCHAR Status Change
started_at DATETIME Start of status/activity
ended_at DATETIME End of status/activity
comment VARCHAR Change comment
